코로나 예방 캠페인을 장려하는 안드로이드 전용 앱입니다.  <br>
게시판에서 회원끼리 격려와 응원을 할 수 있습니다.   <br>

## 개발 환경
언어 : Java<br>
툴 : Android Studio<br>\
DB : Firebase

## 기능
1. 로그인/회원가입/비밀번호 변경
2. 캠페인 실행 여부 체크 기능
3. 게시판
4. 공지사항
